# 🛒 Madhav E-commerce Sales Dashboard

## 📊 Overview

This project is an interactive **E-commerce Sales Dashboard** built to analyze business performance, customer behavior, and profitability using data visualization tools.

It provides insights into sales trends, profit distribution, and regional performance for better decision-making.

---

## 🚀 Key Features

* 💰 **KPI Metrics**

  * Total Sales Amount
  * Total Profit
  * Total Quantity Sold
  * Average Order Value (AOV)

* 📍 **Geographical Analysis**

  * State-wise sales performance (Maharashtra, MP, UP, etc.)

* 📦 **Category Insights**

  * Sales distribution across Clothing, Electronics, Furniture

* 📈 **Profit Trends**

  * Monthly profit visualization to identify seasonal patterns

* 🧾 **Customer Analysis**

  * Top customers by sales contribution

* 💳 **Payment Mode Analysis**

  * COD, UPI, Debit Card, Credit Card, EMI distribution

* 🏷️ **Sub-category Performance**

  * Profit comparison across products (Printers, Saree, Accessories, etc.)

* 🎛️ **Interactive Filters**

  * State filter
  * Quarter filter

## 📂 Dataset

The dataset contains:

* Order details
* Sales amount & profit
* Product categories & sub-categories
* Customer information
* Payment modes
* State-wise transactions

---

## 💡 Key Insights

* Maharashtra generates the highest sales
* Clothing category dominates (~60%+ share)
* COD is the most preferred payment method
* Profit shows fluctuations with losses in some months
* Printers and Bookcases are top profit-generating sub-categories

---

## ⚙️ How to Use

1. Download the `.pbix` file from the `dashboard/` folder
2. Open using Power BI Desktop
3. Interact with filters (State, Quarter) to explore insights
